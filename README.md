👥 Amigo Secreto Programado
Uma aplicação web prática e divertida para organizar sorteios de amigo secreto, focada na experiência do usuário e na integridade dos dados.
🎯 Objetivo do Projeto
O foco principal foi o trabalho com Estruturas de Dados (Arrays) em JavaScript, garantindo que a lista de participantes seja gerida corretamente antes da execução da lógica de sorteio aleatório.
💡 Funcionalidades Principais
• Adição Dinâmica: Permite inserir nomes na lista com validação para campos vazios.
• Validação de Duplicados: Sistema que impede a inserção de nomes repetidos, garantindo a unicidade dos participantes.
• Sorteio Aleatório: Lógica de embaralhamento que seleciona um "amigo secreto" de forma imparcial.
• Interface Intuitiva: Feedback visual imediato ao adicionar nomes ou realizar o sorteio.
🛠️ Tecnologias Utilizadas
• JavaScript: Lógica de manipulação de arrays e eventos de clique.
• HTML5: Estrutura para entrada de dados e exibição de resultados.
• CSS3: Estilização moderna e responsiva.
• GitHub Pages: Hospedagem direta para uso online.
